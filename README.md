# 🤖 AI Powered Twitter Sentiment Analysis

## 📌 Overview

This project is a Machine Learning and Natural Language Processing (NLP) application that predicts the sentiment of tweets as **Positive**, **Negative**, or **Neutral**.

The project uses TF-IDF Vectorization and Logistic Regression to classify tweets and provides an interactive web interface built using Streamlit.

---

## ✨ Features

- Predict tweet sentiment
- Confidence score for predictions
- Dashboard with sentiment statistics
- Pie chart visualization
- Bar chart visualization
- Modular project structure
- Streamlit web application

---
